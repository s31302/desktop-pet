# 🐾 Desktop Pet

## 🚀 Jak pobrać?

1. Przejdź do zakładki **Releases** (po prawej stronie repozytorium).
2. Pobierz najnowszy plik `desktop-pet-v1.zip`.
3. Wypakuj zawartość archiwum w dowolne miejsce na komputerze.
4. Uruchom plik `pet.exe` i ciesz się nowym zwierzakiem na pulpicie.

### ⚙️ Automatyczne włączanie z systemem
Jeśli chcesz, żeby zwierzak budził się razem z komputerem:
* Wypakuj pobrany folder.
* Kliknij prawym przyciskiem myszy na plik `dodaj-auto-start.ps1`.
* Wybierz opcję **Uruchom w programie PowerShell**
